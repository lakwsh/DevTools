# DevTools-lakwsh
一个自带简易混淆功能的PM开发者工具<br>
插件打包指令: `makeplugin *`<br>
服务器打包指令: `makeserver`<br>

## 注意
插件打包指令: `makeplugin <plugin name>`<br>
执行过程中默认会对源代码进行混淆工作<br>
为避免不必要的麻烦,请先备份源代码<br>
